A KEA project to be solved using NEXT JS and REACT.

This project was provided by the agency Charlie Tango and had a duration of two weeks.

Visit site EDC

Introduction

EDC wanted to build a feature so home sellers could test the market and see a glimpse of the potential buyers.

Mechanics

By adding information about their property (zip code, size, type of estate), they get a list of potential buyers that would be interested in a property like that.

Supabase

After selecting different potential buyers, the seller fills up a form that will be sent to EDC, so an employee can contact them. In reality, the seller is sending a post request to Supabase and inserting the information.

Dashboard

The last part of this project was to display the dashboard the EDC employee would see, with all the different insertions made by different sellers.
